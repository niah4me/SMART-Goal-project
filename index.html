My SMART Goal Project 2025

BY: Saniah Cortez
S
Specific

Specific: I want to improve my ability to create smooth, interactive animations using JavaScript and apply CSS media queries to make web designs responsive.

A
Attainable

Achievable: I will dedicate 6–8 hours per week to focused learning and hands-on practice using tutorials, documentation, and code challenges from sources like MDN and CSS-Tricks.

R
Relevant

Relevant: These skills are essential for creating more engaging, accessible, and mobile-friendly web applications—key capabilities for a front-end developer.

T
Time-Based

Time-bound: I will achieve this goal within 3 weeks, setting milestones at the end of each week to track progress and make any necessary adjustments.

M
Measurable

Measurable: I will complete two small JavaScript animation projects and apply media queries to at least one existing or new responsive webpage.
